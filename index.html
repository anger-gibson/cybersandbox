About CyberSandbox
AI-powered static threat analysis for security research and education.
⚠ This tool performs static AI analysis only — it does not execute code, make live network requests to URLs, or detonate files. For live sandboxing, use dedicated tools like Any.run, Cuckoo, or VirusTotal.

Code analysis

Detects reverse shells, obfuscated payloads, persistence mechanisms, credential harvesting, and injection attacks across Python, JS, PowerShell, Bash, PHP, and SQL.

URL / phishing detection

Identifies typosquatting, homograph attacks, suspicious redirect chains, credential harvest pages, and social engineering patterns.

File & malware analysis

Classifies malware families, identifies IOCs, registry persistence, C2 patterns, dropper behavior, and suspicious string indicators from file metadata or contents.

Structured output

Every analysis returns a threat score, severity-rated findings, indicators of compromise, MITRE ATT&CK tags, and step-by-step remediation guidance.

Privacy

Your API key is stored only in your browser's localStorage. All analysis requests go directly from your browser to Anthropic's API — no data is stored on any third-party server.

Hosting this app

  • Drop index.html into any static hosting provider
  • Works on Netlify, GitHub Pages, Vercel, Cloudflare Pages, or any web server
  • No backend required — all API calls are made client-side
  • Each user provides their own Anthropic API key via Settings
